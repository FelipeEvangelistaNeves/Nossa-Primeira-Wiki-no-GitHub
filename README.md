Exemplo:

pwd

Para que serve:
Cria uma nova pasta.

Exemplo:

Para que serve:

Exemplo:

Para que serve:

Exemplo:

📂 7. cp

Para que serve:
Copia arquivos ou pastas.

Exemplo:

cp arquivo.txt copia.txt

Para que serve:
Move ou renomeia arquivos e pastas.

Exemplo:

mv arquivo.txt novo_nome.txt

Para que serve:

Exemplo:

Para que serve:

Exemplo:

Para que serve:
Mostra o conteúdo de um arquivo.

Exemplo:

cat arquivo.txt

Para que serve:
Permite visualizar arquivos grandes página por página.

Exemplo:

less arquivo.txt

Para que serve:

Exemplo:
