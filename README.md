# Nossa-Primeira-Wiki-no-GitHub

🔍 16. grep

Para que serve:
Busca palavras dentro de arquivos.

Exemplo:

grep "erro" arquivo.txt
🔍 17. wc

Para que serve:
Conta linhas, palavras e caracteres de um arquivo.

Exemplo:

wc arquivo.txt
🔍 18. sort

Para que serve:
Organiza o conteúdo em ordem.

Exemplo:

sort lista.txt
🔍 19. uniq

Para que serve:
Remove linhas repetidas.

Exemplo:

uniq lista.txt
⚙️ 20. top

Para que serve:
Mostra os processos em execução em tempo real.

Exemplo:

top
⚙️ 21. htop

Para que serve:
Versão mais visual do top.

Exemplo:

htop
⚙️ 22. ps

Para que serve:
Lista processos em execução.

Exemplo:

ps aux

Para que serve:

kill 1234
